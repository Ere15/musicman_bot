# 🎵 MusicManBot

Асинхронный музыкальный Discord-бот на Python, работающий на базе `discord.py` и `wavelink` (Lavalink v4). 
Для управления зависимостями используется быстрый менеджер пакетов `uv`.

## 🛠 Требования

- **Python** 3.13+
- **Менеджер пакетов uv** (для запуска окружения)
- **Java** 17+ (для работы аудио-сервера Lavalink)
- Утилита `netcat` (`nc`) — используется скриптом для проверки доступности портов

## 📦 Быстрый старт

1. **Перейдите в папку проекта:**
   ```bash
   cd musicmanbot
   ```

2. **Настройте Lavalink:**
   Убедитесь, что в подпапке `lavalink/` лежат файлы `Lavalink.jar` и ваш конфигурационный файл `application.yml`.

3. **Разрешите запуск скрипта автоматизации:**
   ```bash
   chmod +x start.sh
   ```

## 🚀 Запуск проекта

Скрипт автоматически запустит музыкальный сервер Lavalink, дождется его инициализации и поднимет бота:
```bash
./start.sh
```
*Чтобы остановить всю связку, просто нажмите `Ctrl + C` в терминале.*

## 🎮 Команды бота

- `!play <название или ссылка>` — Подключить бота к каналу и запустить трек (YouTube/SoundCloud). Автоматически выравнивает плоский звук через встроенный эквалайзер (BassBoost).
- `!stop` — Полностью остановить воспроизведение аудио и отключить бота от голосового канала.
